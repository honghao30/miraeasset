애니메이션(animation)
애니메이션 가이드이며 JS에서 실행 시점을 제어해야 한다. 
            .slidedown {animation:slidedown 1s ease-out both;}
            @keyframes slidedown {
                0% {opacity:0; transform:translateY(70px);}
                100% {opacity:1; transform:translateY(0px);}
            }
* 미리보기 상태의 코드는 예시이며 사용시 .css or .js 에 내용 참고 바람.
